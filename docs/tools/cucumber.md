<p>
<b>Cucumber</b> es una herramienta es utilizada para implementar metodologías como 
<b>BDD</b> (<b>Behaviour Driven Development</b>). Con ella se puede ejecutar <b>descripciones
funcionales</b> que han sido escritas en <b>texto plano</b> como <b>pruebas de software automatizadas</b>.
En resumen, se describen los procesos de nuestro producto en lenguaje entendible para 
usuarios menos técnicos como el <b>Product Owner</b>, el <b>cliente</b> (<b>negocio</b>) 
o para la incorporación de nuevo personal. Y con este modelo de colaboración 
se ejecutarán de forma automazada cada una de esos casos de pruebas descritos, 
tras un proceso de generación de casos.
</p>
