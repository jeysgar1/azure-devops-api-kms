<p>
<b>Github</b> es un portal creado para <b>alojar el código de las aplicaciones de 
cualquier desarrollador</b>, y que fue comprada por <b>Microsoft</b> en junio del 2018. 
La plataforma está creada para que los desarrolladores suban el código de sus
aplicaciones y herramientas, y que como usuario no sólo puedas descargarte la aplicación, 
sino también entrar a su perfil para leer sobre ella o colaborar con su desarrollo.</b>.
