<p>
<b>NPM</b>  (<b>Node Package Manager</b>) es un <b>gestor de paquetes de NodeJS</b>, 
un entorno de ejecución multiplataforma para ejecutar <b>Javascript</b> no sólo en un navegador web (como se concibió originalmente) 
sino fuera de él, y poder utilizarlo en sistemas de escritorio o servidores web.
</p>
